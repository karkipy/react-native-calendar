# react-native-calendar


## Installation

```javascript
  yarn add @sharingapples/react-native-calendar
```

## Usage

```javascript
  import Calendar from '@sharingapples/react-native-calendar';

  function App() {
    return (
      <Calendar />
    )
  }
```
